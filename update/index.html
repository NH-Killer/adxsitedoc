How to update ANDRAX?

The new generation of ANDRAX is based on Debian, ANDRAX has APT installed but source.list is disabled by default...

Why? Simple, APT is not to be used in ANDRAX, ANDRAX is a Ethical Hacking and Penetration Testing Platform, is not made to common use... you can unlock the sources of apt and install things from Debian repo, but never try update from APT.

Don't use APT!!!

Never use APT to update ANDRAX, this will break the entire system!

Only update ANDRAX from our Official WebSite, every month has new BUILD's with new features and updates.

We are creating a Package Manager!!!

We are working on our own Package Manager, soon it will be ready, for now wait and update using our Official WebSite
